## Project Overview
This project automates the process of sending daily email reports.  
Data is fetched from **Google Sheets**, filtered based on conditions, and emails are sent automatically via **Gmail** on a scheduled basis.

Project Overview
The workflow runs on a daily schedule and performs the following tasks:

workflow steps
1. Triggers automatically using Schedule Trigger
2. Reads rows from Google Sheets
3. Filters data based on defined conditions
4. Uses a Switch node to categorize data
5. Sends automated emails using Gmail
This automation helps reduce manual work and ensures timely reporting.
